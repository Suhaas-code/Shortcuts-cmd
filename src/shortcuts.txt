// Comment (never shown). "# Section" = header, "key<TAB>description" = row.
// `backtick` a key to color it. **bold** *italic* _italic_ = emphasis. "---" = rule.
// "!!!" fences a plaintext block (passwords, IPs, pasted text) — no parsing,
// one color. "!line" does the same for a single line.
// "// ansi = off" disables all color/styling (for SSH/WSL that leak codes).
// "// color <header|key|desc|code> = <spec>" sets colors (see README).
// color header = bright-cyan
// color key    = dim cyan
// color desc   = default
// color code   = cyan

# Panes
`Alt` + `Shift` + `+`	Split pane
`Alt` + `Shift` + `-`	Split pane horizontally
`Alt` + `Shift` + `D`	Duplicate current pane
`Alt` + `Arrow Keys`	Move focus between panes
`Alt` + `Shift` + `Arrow Keys`	Resize current pane
`Ctrl` + `Shift` + `W`	Close current pane

# Search
`Ctrl` + `Shift` + `F`	Search terminal output
`Enter`	Next search result
`Shift` + `Enter`	Previous search result

# Selection
`Ctrl` + `Shift` + `A`	Select all output
`Shift` + `Arrow Keys`	Extend selection
`Alt` + `Drag`	Column selection

# Scrolling
`Ctrl` + `Shift` + `Up`/`Down`	Scroll one line
`Ctrl` + `Shift` + `PgUp`/`PgDn`	Scroll one page
`Ctrl` + `Home`	Scroll to top
`Ctrl` + `End`	Scroll to bottom

# Tabs
`Ctrl` + `Shift` + `T`	New tab
`Ctrl` + `Shift` + `W`	Close current tab
`Ctrl` + `Tab`	Next tab
`Ctrl` + `Shift` + `Tab`	Previous tab
`Ctrl` + `PgUp`	Previous tab
`Ctrl` + `PgDn`	Next tab

# Copy & Paste
`Ctrl` + `Shift` + `C`	Copy
`Ctrl` + `Shift` + `V`	Paste
`Ctrl` + `C`	Interrupt / Copy if selected

# Font Size
`Ctrl` + `+`	Increase font size
`Ctrl` + `-`	Decrease font size
`Ctrl` + `0`	Reset font size

# Window
`Alt` + `Enter`	Fullscreen
`Ctrl` + `Shift` + `P`	Command Palette
`Ctrl` + `,`	Settings

# Command Line
`Tab`	Auto-complete
`Up` / `Down`	Command history
`Home` / `End`	Beginning / End of line
`Ctrl` + `Left`/`Right`	Move by word
`Ctrl` + `Backspace`	Delete previous word
`Ctrl` + `Delete`	Delete next word
`Ctrl` + `L`	Clear screen
`Ctrl` + `R`	Search history

# Notes
!!!
Paste anything here — passwords, IPs, long prompts — and it prints as-is,
in one color, with no `key`/description split and no **bold**/_italic_ parsing.
!!!
!Wrap a single line in "!" the same way, e.g. a lone API key or IP address.
